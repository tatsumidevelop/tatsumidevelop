
【Laravel】開発環境構築 Windows編
https://chigusa-web.com/blog/laravel-setup/
XAMPP,Composer,Node.jsをインストール
MySQLAdminでユーザー、DBを作成

Laravel Framework 8.23.1

cmd.exeからc:\User\tatsumiking d:\laravel-projectにして下記を実行
d:
cd laravel-project
composer create-project laravel/laravel sample-project --prefer-dist

vscode -> Open Folder -> sample-project
 View→Terminal
  php artisan -V // Laravelのバージョンを確認
  npm install // node.jsのモジュールもインストール
  php artisan serve
　　エラー時 sample-projectの下にserver.phpがない場合

Webブラウザーで　[http://127.0.0.1:8000/]　を実行
 Laravelのロゴが出ればOK

XAMPPのMySQLのAdminボタンをクリック
　データベース
　　DBを作成 「utf8mb4_general_ci」を選択
　　　laravel-test
　ユーザーアカウント
　　ユーザーアカウントを追加する　
　　　test 0000
　　　データと構造を全チェック

sample-projectの下の.envを変更
　DB_DATABASE=作成したデータベース名
　DB_USERNAME=設定したユーザ名
　DB_PASSWORD=設定したパスワード

vscode→View→Terminal
　php artisan migrate

XAMPPのMySQLのAdminでDBにテーブルが追加されていればOK
